💻 Basic Computer Notes 1. What is a Computer? A computer is an electronic device that takes input, processes it, and produces output. 2. Main Components of a Computer 🧠 CPU (Central Processing Unit) • Known as the brain of the computer • Performs calculations and processes data 🖥️ Input Devices Used to enter data: • Keyboard • Mouse • Scanner 📺 Output Devices Show results: • Monitor • Printer • Speakers 💾 Storage Devices Store data: • Hard Disk • Pen Drive • CD/DVD 3. Types of Computers • Desktop – Used at home/office • Laptop – Portable computer • Tablet – Touchscreen device • Supercomputer – Very powerful (used in research) 4. Hardware vs Software 🔧 Hardware Physical parts of a computer (keyboard, monitor) 🧩 Software Programs that run on a computer: • System software (Operating System) • Application software (Word, Excel) 5. Operating System (OS) • Controls the computer • Examples: o Windows operating system o macOS o Linux 6. Basic Functions of Computer 1. Input 2. Processing 3. Output 4. Storage 7. Uses of Computers • Education • Business • Banking • Entertainment • Communication
📊 MS Office (Microsoft Office) Notes1. What is MS Office? MS Office is a collection of application software developed by Microsoft used for office, school, and personal work. 2. Main Components of MS Office 📝 Microsoft Word • Used for typing documents • Features: o Text formatting (bold, italic, underline) o Insert images, tables o Spell check 📊 Microsoft Excel • Used for calculations and data management • Features: o Formulas and functions (SUM, AVERAGE) o Charts and graphs o Tables 📽️ Microsoft PowerPoint • Used to create presentations • Features: o Slides o Animations and transitions o Insert images, videos 📧 Microsoft Outlook • Used for sending/receiving emails • Manages calendar, contacts 🗄️ Microsoft Access • Used to store and manage data in databases • Tables, queries, forms, reports 3. Features of MS Office • User-friendly interface • Supports multiple file formats • Easy editing and formatting • Integration between apps 4. Advantages • Saves time and effort • Improves productivity • Easy data management • Widely used in offices and schools 5. Basic File Extensions • Word → .doc / .docx • Excel → .xls / .xlsx • PowerPoint → .ppt / .pptx 6. Uses of MS Office • Creating documents • Data analysis • Presentations • Email communicatio
💰 Tally Notes (Basics) 1. What is Tally? Tally is an accounting software used to record financial transactions of a business. The most popular version is TallyPrime. 2. Features of Tally • Easy accounting • GST calculation • Inventory management • Payroll management • Banking features 3. Important Accounting Terms 📘 Assets • Things owned by a business • Example: Cash, Furniture, Building 📕 Liabilities • Amounts owed by a business • Example: Loans, Creditors 📗 Capital • Owner’s investment in the business 📙 Revenue • Income earned by business 📒 Expenses • Money spent in business 4. Types of Accounts 1. Personal Account o Related to persons or organizations o Example: Ram, Bank 2. Real Account o Related to assets o Example: Cash, Machinery 3. Nominal Account o Related to income & expenses o Example: Salary, Rent 5. Golden Rules of Accounting • Personal Account: 👉 Debit the receiver, Credit the giver • Real Account: 👉 Debit what comes in, Credit what goes out • Nominal Account: 👉 Debit all expenses & losses, Credit all income & gains 6. Basic Steps in Tally 1. Create Company 2. Create Ledger 3. Create Voucher 4. Record Transactions 7. Types of Vouchers • Contra Voucher (F4) • Payment Voucher (F5) • Receipt Voucher (F6) • Journal Voucher (F7) • Sales Voucher (F8) • Purchase Voucher (F9) 8. Advantages of Tally • Saves time • Reduces errors • Automatic calculations • Easy report generation 9. Uses of Tally • Business accounting • GST billing • Financial reports • Inventory tracking
🖨️ DTP (Desktop Publishing) Notes 1. What is DTP? Desktop Publishing (DTP) is the process of creating printed documents like books, magazines, brochures, and posters using a computer. 2. DTP Software Some popular DTP software: • Adobe PageMaker • Adobe InDesign • CorelDRAW • Microsoft Publisher 3. Uses of DTP • Creating books and newspapers • Designing brochures and pamphlets • Making visiting cards • Posters and advertisements 4. Basic Elements of DTP 🅰️ Text • Written content in the document 🖼️ Graphics • Images, logos, designs 📐 Layout • Arrangement of text and images 🔤 Fonts • Style and size of text 5. Important Features • Page layout design • Text formatting • Image editing • Print preview • Templates 6. Advantages of DTP • Professional design • Saves time and cost • Easy editing • High-quality printing 7. Common DTP Terms • Alignment – Position of text (left, right, center) • Margins – Space around page • Columns – Dividing page into sections • Spacing – Distance between lines/letters 8. Output Devices Used in DTP • Printer • Plotter 9. Applications of DTP • Publishing industry • Advertising agencies • Printing press • Education
💻 Programming Notes (Basics)1. What is Programming? Programming is the process of writing instructions (code) that a computer can understand and execute. 2. What is a Program? A program is a set of instructions written to perform a specific task. 3. Programming Languages Languages used to write programs: • C • C++ • Java • Python 4. Types of Programming Languages 🔹 Low-Level Language • Machine language, Assembly language • Difficult to understand 🔹 High-Level Language • Easy to read and write • Example: C, Java, Python 5. Translator Programs • Compiler – Converts whole program at once • Interpreter – Converts line by line 6. Basic Concepts 🧮 Algorithm • Step-by-step solution to a problem 📊 Flowchart • Diagram representing steps of an algorithm 🔤 Variable • Stores data value 🔢 Data Types • Integer (int) • Float (decimal) • Character (char) 7. Control Structures 🔁 Looping • Repeats code • for, while 🔀 Decision Making • if, else 8. Functions • Block of code that performs a specific task • Helps reuse code 9. Errors in Programming • Syntax Error – Wrong code format • Logical Error – Wrong logic • Runtime Error – Error during execution 10. Advantages of Programming • Automates tasks • Solves complex problems • Builds software and apps 11. Uses of Programming • Software development • Web development • Game development • Data analysis
